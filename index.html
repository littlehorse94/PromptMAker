How many pages?

Our recommendation: 5 pages (Home, About, Services, Blog, Contact) is the sweet spot for most SMEs — enough depth without overwhelming visitors or delaying launch.

Dynamic website with CMS
Build a database / content management system so you can update products, blog posts, team members etc. without a developer
Multilingual website
Support multiple languages. Note: each additional language increases development time and potential translation errors
⚠️ Heads up: The more languages you add, the longer the project takes and the higher the chance of translation inconsistencies. We recommend starting with 2 max.

SEO &
Performance

Choose how much SEO optimisation you want baked into your website from day one.

⚡ Full SEO

Complete Optimisation

Everything you need to rank. Includes GA4 and Google Search Console setup guidance, plus every on-page technical element.

  • Google Analytics 4 setup
  • Google Search Console integration
  • XML Sitemap generation
  • robots.txt & llms.txt
  • Meta titles & descriptions (all pages)
  • Open Graph / social sharing tags
  • Schema.org structured data
  • Alt text for all images
  • Canonical tags
  • H1–H6 hierarchy
  • Keyword mapping per page
  • Core Web Vitals optimisation
🔶 Half SEO

Essential Optimisation

The core on-page elements to make your site search-friendly without the analytics infrastructure setup.

  • Meta titles & descriptions
  • Open Graph tags
  • Schema.org markup
  • Page excerpts / summaries
  • Alt text for key images
  • Basic H1–H3 structure
  • GA4 / GSC setup
  • Sitemap / robots.txt
  • llms.txt
⬜ No SEO

Skip for Now

Focus entirely on the design and functionality. SEO can be added in a later phase of the project.

  • Fully designed pages
  • No SEO elements added
  • Faster initial build
  • All SEO features
